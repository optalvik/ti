## Sihtrühm

Kursus on mõeldud IT-süsteemide nooremspetsialisti (tase 4) õppekava õpilastele, kes soovivad omandada tehisintellekti tööriistade praktilisi oskusi. Eelteadmisi AI valdkonnas ei eeldata.

## Eeldused

- Arvuti algteadmised ja internetiühendus
- Terminali/käsurea põhioskused (Ollama ja Claude Code praktikumide jaoks)
- Huvi tehnoloogia vastu

---

## Õpiväljundid

Mooduli läbinu:

1. **seostab** tehisintellekti kasutusvõimalusi ja selleks vajalikke eeldusi digitehnoloogias,
2. **on kursis** klassikaliste algoritmide ja andmestruktuuride kasutamise võimalustega tehisintellekti rakendamisel,
3. **oskab formuleerida** klassikalisi otsinguülesandeid ja rakendada masinõppe elementaarseid tehnikaid.

| 2. Algoritmid ja andmestruktuurid AI-s | 2, 8 | P5 |

## Kursuse ülesehitus

Kursus koosneb 10 teooriaosast ja 6 praktikumist.

**Teooria** katab tehisintellekti ajaloo ja põhimõisted, masinõppe alused, suurte keelemudelite tööpõhimõtted, prompt engineering tehnikad, AI tööriistade ülevaate, koodi genereerimise, agentliku töövoo, andmed ja algoritmid AI kontekstis, eetika ja turvalisuse ning tulevikutrendid.

**Praktikumid** juhendavad samm-sammult LLM-ide kasutamist: vestlusrobotid, promptimine, Ollama lokaalne paigaldamine, Claude Code CLI, API päringud ja agentlik töövoog MCP serveritega.

## Kasutusjuhend

1. Alusta [teooria peatükkidega](teooria/index.md) järjest — iga peatükk viitab eelmisele
2. Pärast iga teooriaosa lõpetamist vasta enesekontrolliküsimustele
3. Tee [praktikumid](praktikumid/index.md) vastavas järjekorras
4. Kasuta otsinguvälja (++ctrl+k++) kiireks navigeerimiseks

!!! tip "Tumeda režiimi lülitamine"
    Kasuta lehe ülaosas olevat päikese/kuu ikooni tumeda ja heleda teema vahel lülitamiseks.
